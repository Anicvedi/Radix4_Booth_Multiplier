# Booth Radix-4 Multiplier

## Overview

Verilog implementation of a Booth Radix-4 multiplier for efficient signed multiplication.

## Features

* Reduced partial products
* Signed multiplication
* Hardware-efficient design

## Structure

* rtl/ → design files
* tb/ → testbench
* results/ → waveforms

## How to Run

Simulate using ModelSim/Vivado.
